# Project Overview

**Clash Verge Rev** is a cross-platform desktop GUI application for managing Clash Meta (mihomo) proxy configurations.

## Purpose

- Provides a modern, user-friendly interface for network proxy management
- Built as a continuation of the original Clash Verge project
- Supports Windows (x64/x86), Linux (x64/arm64), and macOS 10.15+ (intel/apple)
- Manages Clash Meta proxy configurations with visual node and rule editing

## Key Features

- Configuration file management with merge and script enhancements
- System proxy and TUN (virtual network card) mode support
- Visual proxy group and rule editing
- WebDAV configuration backup and synchronization
- Custom theme support with CSS injection
- Multi-language interface (i18next)
- Built-in Clash.Meta (mihomo) kernel with Alpha version support
